# hctrackr

<!-- badges: start -->

<!-- badges: end -->

The goal of hctrackr is to provide tools and standards for specifying
trail and target hiking challenges as well as assessing and recording
completion of those challenges.

## Installation

You can install the development version of hctrackr from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("ryanraaum/hctrackr")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(hctrackr)
## basic example code
```
